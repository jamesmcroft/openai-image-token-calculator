# Contributing

These guidelines are designed to help you as a contributor to bring your ideas and skills to the Azure OpenAI Image Token Calculator project.

## What can I contribute to?

Our [issue tracker](https://github.com/jamesmcroft/openai-image-token-calculator/issues) contains a collection of features and issues waiting to be worked on! If you see any item marked with **Help Wanted**, we'd like to see you put your skills to use and contribute whatever you can.

This can be in the form of contributing to the conversation around a feature or issue, providing code and submitting a pull request, adding a new feature request or bug, or even helping to update our documentation.

We know that not everyone can do everything, so any little helps. We accept any contribution whether big or small.

## Running Locally

1. Ensure you have [Node.js](https://nodejs.org/) installed (LTS recommended).
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
4. Open `http://localhost:5173` in your browser.

## Code Styling

The project uses ESLint for linting. Run `npm run lint` to check for issues. We're happy for you to code in a way that makes you feel comfortable, and we'll work together to make sure everything is consistent before merging.

## Bug Hunting

We're more than happy for you to contribute through testing. Finding bugs is definitely encouraged. Please remember to start a new issue in our [issue tracker](https://github.com/jamesmcroft/openai-image-token-calculator/issues) so we can keep an eye on them and get bugs fixed.

Even better, if you know how to fix it, follow up your issue with a pull request!

## Documentation

If you find something that is wrong or missing in the documentation, feel free to raise an issue or contribute towards improving it.
